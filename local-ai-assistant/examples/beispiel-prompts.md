# Beispiel-Prompts

Hier findest du Beispiel-Prompts fuer den KI-Assistenten, gruppiert nach Plugin/Server.

## Outlook (E-Mail)

- "Fasse mir alle E-Mails von heute zusammen"
- "Suche nach E-Mails zum Thema Deployment"
- "Was steht in der letzten E-Mail von Max Mueller?"
- "Welche ungelesenen E-Mails habe ich?"
- "Liste alle meine E-Mail-Ordner auf"

## Dateisystem

- "Lies die Datei C:\Users\ich\Documents\notizen.txt"
- "Erstelle eine Zusammenfassung aller .md Dateien in meinem Projekte-Ordner"
- "Suche nach Dateien mit dem Namen 'Rechnung' in meinen Dokumenten"
- "Zeige mir die Verzeichnisstruktur von meinem Projekte-Ordner"

## Kombinierte Anfragen (mehrere Plugins)

- "Suche in meinen E-Mails nach dem Projektbericht und speichere eine Zusammenfassung als Textdatei"
- "Lies die Datei Aufgaben.txt und erstelle daraus eine E-Mail an mein Team"

## Memory (wenn aktiviert)

- "Merke dir: Mein aktuelles Projekt heisst Phoenix und der Deadline ist der 15. April"
- "Was habe ich dir letztens ueber das Phoenix-Projekt erzaehlt?"

## Everything Search (wenn aktiviert)

- "Finde alle PDF-Dateien die das Wort 'Vertrag' im Namen haben"
- "Wo liegt die Datei 'praesentation_q4.pptx' auf meinem PC?"

## Fetch (wenn aktiviert)

- "Rufe die Webseite https://example.com ab und fasse den Inhalt zusammen"
- "Was steht auf der Startseite von heise.de?"

## SQLite (wenn aktiviert)

- "Zeige mir alle Tabellen in der Datenbank"
- "Fuehre eine SQL-Abfrage auf die Tabelle 'kunden' aus: SELECT * FROM kunden LIMIT 10"

## Tipps fuer gute Prompts

- **Sei spezifisch:** "Suche E-Mails von Max Mueller zum Thema Deployment von letzter Woche" statt "Zeig mir E-Mails"
- **Kombiniere Plugins:** "Lies meine letzte E-Mail von Max und speichere den Anhang-Link in einer Textdatei"
- **Nutze Kontext:** "Basierend auf den E-Mails von heute, erstelle eine Aufgabenliste"
